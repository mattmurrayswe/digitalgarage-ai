# digitalgarage-ai — Local URLs

Use these on your MacBook while Docker is running.

- Frontend (home): http://localhost:3000
- Frontend (listings): http://localhost:3000/listings
- Backend health: http://localhost:8000/health
- Backend listings API: http://localhost:8000/listings?limit=20

## Quick start

From project root:

```bash
docker compose up -d --build
```

Then open the URLs above.
